Highway Stations Planner

Implemented in C, this project models highway service stations with a BST and per-station vehicle lists. It supports dynamic updates and computes feasible routes based on vehicle autonomy.

Overview

Stations are stored in a binary search tree keyed by distance from the highway origin. Each station keeps a sorted doubly linked list of vehicles, allowing fast access to the max autonomy car for routing.

Core operations

  • Add or demolish stations in the BST.
  • Add or scrap vehicles while preserving sorted autonomy.
  • Plan routes by chaining reachable stations.
  • Stream command I/O for batch processing.
