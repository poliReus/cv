Overview

The project delivers a full playable version of Codex Naturalis with robust networking and game management. The server supports multiple concurrent games, chat channels, and reconnection logic.

Key features

  • Multiple simultaneous games with server-side orchestration.
  • In-game chat (private and global).
  • Reconnect support for dropped players.
  • Choice of CLI or GUI client.
